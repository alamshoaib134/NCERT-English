# 📖 Poorvi Reader — NCERT Class 7 Bilingual Textbook

![Poorvi Reader Banner](public/images/ch1_jahnavi_river.png)

A modern, mobile-first web application designed to make learning English accessible and engaging for Class 7 students reading the NCERT *Poorvi* textbook. Built with Next.js, Tailwind CSS, and TypeScript.

---

## 🎯 Purpose and Motivation

**Why did we start this?**
Learning a new language can be daunting, especially for students who may not have a strong foundation in English or lack access to immediate teacher support. Traditional textbooks are static; if a student doesn't understand a word or a sentence, they often get stuck, leading to frustration and cognitive fatigue. 

We built **Poorvi Reader** to bridge this gap. By turning the static textbook into an interactive, bilingual digital experience, we aim to empower students to learn independently, confidently, and at their own pace.

**Our Core Goals:**
- **Reduce Cognitive Load:** Presenting English text alongside immediate Hindi translations in a visually distinct, soothing format.
- **Instant Support:** Providing an interactive dictionary where tapping any word instantly brings up its meaning and pronunciation.
- **Accessibility:** Creating a UI so simple and intuitive that even tech-unfamiliar users can navigate it without a learning curve.

## 🧑‍🤝‍🧑 Who are the Users?

1. **Class 7 Students (Primary Users):** Specifically, students in India studying the NCERT curriculum who are transitioning to more complex English texts. The bilingual support (English + Hindi) is tailored to help native Hindi speakers grasp English nuances comfortably.
2. **Teachers & Educators:** Teachers who want a digital, interactive aid to use in classrooms or to assign for homework. The built-in comprehensive Q&A sections make it a great tool for revision.
3. **Parents:** Parents who want to help their children study at home but might not be completely fluent in English themselves. The bilingual nature of the app allows parents to follow along and guide their children effectively.

## ✨ Key Features

- **📱 Mobile-First UX:** Designed primarily for mobile screens (where most students will access it), featuring simple tap mechanics and no complex menus.
- **🌍 Split-View Layout:** Every paragraph is presented in a soft card block with the English sentence prominent and the Hindi translation directly underneath in a soothing, distinctly colored text box.
- **📖 Interactive Dictionary:** Tapping any English word brings up a clean bottom sheet with its meaning (fetched dynamically or from a local cache).
- **📝 Comprehensive Q&A:** Includes all textbook exercise sections (Let Us Do, Check Your Understanding, Let Us Think and Reflect, Vocabulary, Grammar, and Speaking Activity) in a bilingual, easily navigable format.
- **🎨 Rich Aesthetics:** Features curated color palettes, beautiful AI-generated watercolor illustrations, and smooth micro-animations to create a premium, engaging learning environment.
- **⚡ Fast & Offline-Ready:** Built as a static-exportable Next.js app, designed to be fast and lightweight for varying network conditions.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## 🛠️ Tech Stack

- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS (Customized for a premium, accessible aesthetic)
- **Language:** TypeScript
- **Icons:** Lucide React / Native Emojis
- **Deployment:** Vercel (Static Export compatible)
